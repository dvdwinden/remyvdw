Personal portfolio website for Remy van der Winden, a Motion Designer from the Netherlands.

Built with [Eleventy](https://www.11ty.dev/) and [Tailwind CSS](https://tailwindcss.com/).

## Getting Started

### Prerequisites

- Node.js 20 or higher
- npm

### Clone and Install

```bash
# Clone the repository
git clone https://github.com/yourusername/remyvdw.git
cd remyvdw

# Install dependencies
npm install
```

### Development

```bash
# Start the development server
npm run dev
```

The site will be available at `http://localhost:8080` with live reload enabled.

### Build

```bash
# Build for production
npm run build
```

The built site will be in the `_site/` directory.

## Deployment

The site automatically deploys to GitHub Pages when changes are pushed to the `main` branch.
